# Singly-Linked-List
* A singly linked list is a concrete data structure consisting of a sequence of nodes.
* Each node stores:
  * Element
  * Reference to the next node
* The last node in the list points to a null.
* Here are some important methods in singly linked list:
  * <code>addFirst()</code>
  * <code>addAfter()</code>
  * <code>remove()</code>
